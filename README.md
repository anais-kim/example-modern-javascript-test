# example-modern-javascript-test
Simple javascript app searching books via GoogleBooks API, <br>
testing with ***mocha, babel, chai, sinon, fetch-mock, jsdom, istanbul***

----------------
### Requirements
- node & npm

----------------
### Tech Stack
- node
- ES6
- babel
- webpack
- mocha
- chai
- sinon
- fetch-mock
- jsdom
- istanbul

----------------
### How to run
```javascript
// to run devserver
npm run devserver

// to run unit test
npm run test

// to make test coverage report
npm run coverage
```
